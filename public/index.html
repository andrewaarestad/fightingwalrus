Fighting Walrus Radio

iOS Radio for UAVs



Key Features

  • Integrated 3D Robotics Radio
  • Supports AR Drone, Ardupilot
  • 1 mile approximate range
  • Open Source software
  • All-metal design
  • Supports iPhone and iPad
  • Lightning and 30-pin versions
